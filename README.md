# Qcard Mobile App


Mockup of the app allows to:
- Generate qr code
- Scan qr codes
- check for payments from clients(as a seller)

## How to run

Clone the project

```bash
  git clone https://github.com/JustAnotherDevv/QcardMobileApp
```

Enter directory

```bash
  cd QCard
```

Install dependencies

```yarn
```

Start the expo app

```expo start
```
